# vux2-project

> vue 项目

## 启动配置

``` bash
# 下载依赖
npm install

# 热重载服务在 localhost:5802
npm run dev

# 压缩打包成生产环境
npm run build

# 建立生产环境和查看分析报告
npm run build --report

```

## 内置说明

``` bash
# 数据请求响应:axios
 详情参见:https://github.com/axios/axios

# 伪数据请求拦截
开发环境中:ctrl+alt+鼠标左键

# UI组件: vux 
详情参见:https://vux.li/
```
